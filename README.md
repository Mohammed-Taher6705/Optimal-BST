Implementation of an Optimal Binary Search Tree using dynamic programming in C++. The program minimizes search time based on element access frequencies.

## Overview
An Optimal Binary Search Tree (BST) is a binary tree that minimizes the expected search cost based on the access frequencies of elements. This project implements the algorithm using dynamic programming in C++.

## Features
- Constructs an Optimal BST from a list of keys and their access frequencies.
- Uses dynamic programming for efficient computation.
- Displays the structure of the Optimal BST.
